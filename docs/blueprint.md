# **App Name**: FinFlow

## Core Features:

- Home Screen: Display personalized greeting, total balance, income vs expense stats, recent transactions, and a floating action button (FAB) for adding transactions.
- Add Transaction Modal: A slide-up bottom sheet to add transactions, featuring an amount input, income/expense toggle, category selection grid, date picker, and note input.
- Analytics Dashboard: Interactive analytics dashboard with time-based tabs, spending breakdown donut chart, trend line area chart, and AI-generated insights and summary.
- Transaction History: Display a complete list of transactions, grouped by date, with search and filter options.
- Mobile-First Responsiveness: Ensure full responsiveness and native-app feel across various mobile devices.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke trust and calmness in financial matters.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Muted purple (#BDB2FF) to add a touch of creativity and uniqueness to CTAs.
- Body and headline font: 'Inter', a sans-serif, for a modern, readable, and clean user interface.
- Card-based layout with soft shadows and rounded corners (xl or 2xl) to enhance the modern aesthetic and improve content organization.
- Use 'Lucide-React' icons throughout the app to maintain consistency and provide clear visual cues.
- Use Framer Motion for smooth page transitions, slide-up modals, and subtle button tap animations to enhance user experience.
